You can first ajust the parameters used to generate the accumulation map: the angular resolution (nbAngularSectors), the maximal analysis radius from the pith (rMax) and the minimal Z-Motion value considered (zMin, set to 10 by default remove the background noise).

You can also change the parameters used to detect th supporting areas of knots: the threshold value to binarized the accumulation map (tBin) and the maximum distance between two connected components (dMax, set to 20 by default according the mean width od knots).
